# RetroMarket-frontend
proyecto basado en una landing page de  tienda vintage sobre ropa con para el certamen 1
##lenguajes usados
HTML y CSS (puro)

##caracteristicas
- Catálogo con tarjetas (estados: Disponible / Vendido con variables CSS)
- Buscador y filtros por categoría
- Formulario para publicar prendas
- Carrito lateral con animación de entrada desde la derecha
- Diseño responsivo para móvil y escritorio

###implementamos animaciones(@keyframes)
- `aparecer` — cards entran al cargar la página
- `fadeHero` — portada aparece con efecto de zoom suave
- `pulsar` — botón del catálogo pulsa al hacer hover
